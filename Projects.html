Sistema de Registro de Serviços de Máquinas e Equipamentos para a Secretária de Agricultura de Prefeituras

O sistema permite o registro de serviços realizados com tratores (e seus implementos), retroescavadeiras e caçambas, incluindo detalhes como quantidade de cargas, cascalho e outros materiais. Além disso, possui uma página de visualização onde é possível ver todos os serviços cadastrados. Os serviços podem ser ordenados por localidade, nome e tipo de serviço. Nesta página, há um botão "Concluído", onde o secretário pode marcar o serviço como finalizado e inserir a data de conclusão. Há também uma página dedicada à visualização dos serviços concluídos, que pode ser ordenada por nome.

Sistema de Registro para marcação de viagens da Secretária de Saude de Prefeituras

O sistema de saúde é projetado para agendar viagens, registrando informações como o paciente, data da consulta, veículo, motorista, cidade, nome do passageiro, hora, CPF, telefone, motivo e local da consulta (clínica, hospital). Ele possui uma página para visualização das viagens, onde aparecem detalhes como veículo, destino, motorista, passageiros e seus acompanhantes, com a possibilidade de imprimir os dados da viagem. Além disso, há um relatório de cada viagem realizado pelo motorista, registrando a data de saída e chegada, e relatando quaisquer ocorrências. Cada motorista tem acesso apenas às suas próprias viagens. Existe também uma aba para visualizar viagens concluídas, onde são apresentados todos os dados da viagem juntamente com o relatório do motorista.

Tecnologias utilizadas: PHP, HTML, CSS, JavaScript, Ajax, MySql, arquitetura MVC e o framework bootstrap
